# SQL?

## SQL 언어의 구성
- 데이터 정의어(DDL)
  - CREATE
  - ALTER
  - DROP
- 데이터 조작어(DML)
  - INSERT
  - UPDATE
  - DELETE
  - SELECT
- 데이터 제어어(DCL)
  - GRANT
  - REVOKE
